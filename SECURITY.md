# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please do not open a public GitHub issue.

Use GitHub Security Advisories if private reporting is enabled for this
repository. If private reporting is not available, contact the repository
owner directly through the maintainer profile and include:

- a clear description of the issue
- affected files or components
- reproduction steps
- potential impact

Please allow time for review before disclosing details publicly.
